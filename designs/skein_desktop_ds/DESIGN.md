# Skein Desktop Design System

A technical, noctural design system for the Skein automation platform.

## Principles

- **Nocturnal:** High contrast dark theme for developer focus.
- **Precise:** Monospace accents for technical data.
- **No-Nonsense:** Clean lines, purposeful spacing, and restrained use of color.

## Components

- **Sidebar:** Fixed left navigation with icon + label.
- **Workflow Card:** Grid-based cards with status indicators and run history sparklines.
- **Top Bar:** Contextual information and global search.
